# Salesforce MCP Setup Guide

A step-by-step guide for installing and configuring the Salesforce MCP server on macOS for TELUS Digital employees.

## Quick Start

Open the guide in your browser: [**View the setup guide**](https://raw.githubusercontent.com/makwt/salesforce-mcp-setup-guide/main/index.html)

Or download this repository and open `index.html` in your browser.

## What is Salesforce MCP?

The Salesforce MCP (Model Context Protocol) server exposes Salesforce data—projects, revenue, people, timecards, skills, and assignments—as tools for Claude and other AI assistants. Ask Claude questions about your work in natural language and get real-time Salesforce data back.

## Two Paths for Access

This guide includes **two paths** for downloading the project:

- **Path A**: For those with access to the WillowTree GitHub organization
- **Path B**: For those without WillowTree organization access (uses Mohammed's personal repository at `github.com/makwt/salesforce-mcp`)

- No coding experience needed
